Section 7.5 Exercises

The exercises in Liebeck's book relevant to this section are in Chapter 23.

Checkpoint 7.5.1.

Show that \(|x- L| \lt \epsilon\) if and only if \(L - \epsilon \lt x \lt L+ \epsilon\text{.}\)

Solution.

Suppose that \(|x-L|\lt \epsilon\text{.}\) Then \(x-L\leq |x-L|\lt \epsilon, \mbox{ and so } x \lt L + \epsilon;\) moreover \(L-x \leq |L-x|=|x-L|\lt \epsilon \mbox{ and so } x>L-\epsilon\text{.}\) Conversely, suppose that \(L-\epsilon\lt x\lt L+\epsilon\text{.}\) Then \(- \epsilon \lt x -L \lt \epsilon\) and so \(|x-L|\lt \epsilon\text{.}\)

Checkpoint 7.5.2.

Using the \(\epsilon-N\) definition of a limit, prove the following:

  1. \(\limn \frac{1}{n^{3}}=0\text{.}\) Solution.

    Let \(\epsilon>0\text{.}\) If \(N=\epsilon^{-1/3}\text{,}\) then for \(n> N\text{,}\)

    \begin{equation*} \left|\frac{1}{n^3}-0\right|=\frac{1}{n^3}\lt \frac{1}{N^3}=\frac{1}{ (\epsilon^{-1/3})^3}=\frac{1}{\epsilon^{-1}}=\epsilon\text{.} \end{equation*}

  2. \(\limn \frac{n^{4}}{n^{4}+1}=1\) Solution.

    Let \(\epsilon>0\text{.}\) We want to find \(N\) so that \(n> N\) implies

    \begin{equation*} \left|\frac{n^{4}}{n^{4}+1}-1\right|=\frac{1}{n^4+1}\lt \epsilon\text{.} \end{equation*}

    If \(\epsilon\geq 1\text{,}\) then this always holds so we can just pick \(N=1\text{.}\) If \(\epsilon\lt 1\text{,}\) this is equivalent to saying \(n^4+1>\epsilon^{-1}\text{.}\) This will certainly hold if \(n^4>\epsilon^{-1}\text{,}\) i.e. when \(n>\epsilon^{-\frac{1}{4}}\text{.}\) Thus, the above inequality holds if \(n>N\) where \(N=\epsilon^{-\frac{1}{4}}\text{.}\)

  3. \(\limn \frac{n^{2}+3}{n^{2}+n}=1\) Solution.

    Let \(\epsilon>0\text{,}\) we want to find \(N\) so that \(n> N\) implies

    \begin{equation*} \left|\frac{n^{2}+3}{n^{2}+n}-1\right|=\left|\frac{3-n}{n^2+n}\right|\lt \epsilon\text{.} \end{equation*}

    Note that for \(n> 3\text{,}\) \(n-3> 0\) and so

    \begin{equation*} \left|\frac{3-n}{n^2+n}\right|=\frac{n-3}{n^2+n}\lt \frac{n}{n^2+n}=\frac{1}{n+1}\text{.} \end{equation*}

    Thus, if we pick \(N=\max\{3,\frac{1}{\epsilon}\}\text{,}\) we see that \(n> N\) implies

    \begin{equation*} \left|\frac{n^{2}+3}{n^{2}+n}-1\right|=\frac{n-3}{n^2+n}\lt \frac{1}{n+1}\lt \frac{1}{N+1}\leq \frac{1}{\frac{1}{\epsilon}+1}\lt \frac{1}{1/\epsilon}=\epsilon\text{.} \end{equation*}

  4. \(\limn \frac{2n^{2}-n}{n^{2}+n-1}=2\text{.}\) Solution.

    Let \(\epsilon>0\text{,}\) we want to find \(N\) so that \(n> N\) implies

    \begin{equation*} \left|\frac{2n^{2}-n}{n^{2}+n-1}-2\right| =\left|\frac{-3n+2}{n^2+n-1}\right|\lt \epsilon\text{.} \end{equation*}

    Since \(n\geq 1\text{,}\) \(3n-2\geq 0\text{,}\) so

    \begin{equation*} \left|\frac{-3n+2}{n^2+n-1}\right|=\frac{3n-2}{n^2+n-1} \lt \frac{3n}{n^2-1}=\frac{3n}{(n-1)(n+1)}\lt \frac{3n}{(n-1)n}=\frac{3}{n-1}\text{.} \end{equation*}

    We want \(N\) so that \(n> N\) implies \(\frac{3}{n-1}\lt \epsilon\text{,}\) i.e. that \(n>3\epsilon^{-1}+1\text{.}\) Thus, if we pick \(N=3\epsilon^{-1}+1\text{,}\) then \(n> N\) implies

    \begin{equation*} \left|\frac{2n^{2}-n}{n^{2}+n-1}-2\right|=\frac{3n-2}{n^2+n-1}\lt \frac{3}{n-1}\lt \frac{3}{N-1} =\frac{3}{3\epsilon^{-1}+1-1}=\epsilon\text{.} \end{equation*}

Checkpoint 7.5.3.

For the following sequences, show that they tend to infinity.

  1. \(5n-2\) Solution.

    Let \(M>0\text{.}\) Let \(N>\frac{M+2}{5}\text{,}\) then for \(n> N\text{,}\)

    \begin{equation*} 5n-2> 5N-2\geq 5\frac{M+2}{5}-2=M\text{.} \end{equation*}

  2. \(n^{3}+1\) Solution.

    Note that \(n^3+1>n^3\text{,}\) so if \(M>0\text{,}\) we can pick \(N=M^{\frac{1}{3}}\) and then \(n> N\) will imply

    \begin{equation*} n^3+1>n^3> N^3=M\text{.} \end{equation*}

  3. \(n^{4}-n+1\) Solution.

    Note that for \(n>1\text{,}\)

    \begin{equation*} n^4-n+1=n(n^3-1)+1\geq n+1 \end{equation*}

    and so if \(N=M-1\text{,}\) then for \(n>N-1\text{,}\) we have \(n^4-n+1\geq n+1\geq N+1\geq M\text{.}\)

  4. \(n!\) Solution.

    Let \(M>0\text{.}\) If we set \(N=M\text{,}\) then \(n> N\) implies

    \begin{equation*} n!\geq n> N=M\text{.} \end{equation*}

  5. \(\sqrt{n}\) Solution.

    Let \(M>0\) and \(N=M^2\text{,}\) then \(n> N\) implies

    \begin{equation*} \sqrt{n}> \sqrt{N}=\sqrt{M^2}=M\text{.} \end{equation*}

Checkpoint 7.5.4.

Show that the limit of a sequence is unique, that is, if \(x_{n}\rightarrow L\) and \(x_{n}\rightarrow M\text{,}\) then \(L=M\text{.}\)

Solution.

Suppose \(L\neq M\text{.}\) Let \(\epsilon=\frac{|L-M|}{2}\text{.}\) Since \(x_{n}\rightarrow L\text{,}\) there is \(N_1\) so that \(n> N_1\) implies \(|x_{n}-L|\lt \epsilon\text{.}\) Similarly, \(x_{n}\rightarrow M\text{,}\) there is \(N_2\) so that \(n> N_2\) implies \(|x_{n}-L|\lt \epsilon\text{.}\) Thus, for \(n>N= \max\{N_1,N_2\}\text{,}\)

\begin{equation*} |L-M|=|L-x_{n}+x_{n}-M|\leq |L-x_{n}|+|x_{n}-M|\lt \frac{\epsilon}{2}+\frac{\epsilon}{2} = \epsilon = |L-M| \end{equation*}

which is a contradiction.

Checkpoint 7.5.5.

For each \(L\) below, find sequences \((x_{n})\) and \((y_{n})\) so that \(x_{n}\rightarrow 0\text{,}\) \(y_{n}\rightarrow 0\) and \(x_{n}\neq 0, y_n \neq 0\) for all \(n\text{,}\) and such that \(\frac{x_{n}}{y_{n}}\rightarrow L\text{.}\)

  1. \(\displaystyle 1\)

  2. \(\displaystyle 0\)

  3. \(\displaystyle \infty\)

Solution.

  1. \(x_n=y_n=\frac{1}{n}\text{.}\)

  2. \(x_n=\frac{1}{n^2}, y_n=\frac{1}{n}\text{.}\)

  3. \(x_n=\frac{1}{n}, y_n=\frac{1}{n^2}\text{.}\)

(Many other solutions are possible.)

Checkpoint 7.5.6.

For each \(L\) below, find sequences \((x_{n})\) and \((y_{n})\) so that \(x_{n}\rightarrow 0\text{,}\) \(y_{n}\rightarrow \infty\) and \(x_{n}y_{n}\rightarrow L\text{.}\)

  1. \(\displaystyle 1\)

  2. \(\displaystyle 0\)

  3. \(\displaystyle \infty\)

Solution.

  1. \(x_n=y_n=n\text{.}\)

  2. \(x_n=n, y_n=n^2\text{.}\)

  3. \(x_n=n^2, y_n=n\text{.}\)

Checkpoint 7.5.7.

Show that if \(x_n\rightarrow x\text{,}\) then \(|x_n|\rightarrow |x|\text{.}\)

Solution.

Let \(\epsilon>0\text{.}\) Since \(x_n\rightarrow x\text{,}\) there is \(N\) so that \(n> N\) implies \(|x_{n}-x|\lt \epsilon\text{.}\) By the reverse triangle inequality, for \(n> N\text{,}\)

\begin{equation*} ||x_{n}|-|x|| \leq |x_{n}-x|\lt \epsilon\text{.} \end{equation*}

Thus, \(|x_n|\rightarrow |x|\text{.}\)

Checkpoint 7.5.8.

Suppose that \((x_{n})\) converges. Show that for any \(k\in\mathbb{N}\text{,}\) \(\limn x_{n}= \limn x_{n+k}\text{.}\)

Solution.

Let \(L=\limn x_{n}\text{.}\) We want to show \(\limn x_{n+k}=L\text{,}\) that is, for all \(\epsilon>0\) there is \(N\) so that \(n> N\) implies \(|x_{n+k}-L|\lt \epsilon\text{.}\) Let \(\epsilon>0\text{.}\) Since \(L=\limn x_{n}\text{,}\) we know there is \(N\) so that \(n> N\) implies \(|x_{n}-L|\lt \epsilon\text{.}\) If \(n> N\text{,}\) then \(n+k>N\text{,}\) and so we also have \(|x_{n+k}-L|\lt \epsilon\) for all \(n> N\text{.}\) This proves the claim.

Checkpoint 7.5.9.

Suppose that \((x_{n})\) converges. Show that \(x_{n}-x_{n+1}\rightarrow 0\text{.}\)

Solution.

Using the limit rules and the previous exercise, if \(\limn =L\text{,}\) then

\begin{equation*} \limn(x_{n}-x_{n+1}) =\limn x_n - \limn x_{n+1} = L-L=0\text{.} \end{equation*}
Checkpoint 7.5.10.

Using the \(\epsilon-N\) definition of a limit, prove that if \(x_n>0\) and \(x_n\rightarrow x>0\) and \(x_n\neq x\) for all \(n\text{,}\) then \(x_{n}^{\frac{1}{3}}\rightarrow x^{\frac{1}{3}}\text{.}\)

Solution.

Note that

\begin{equation*} |x^{\frac{1}{3}}-x_{n}^{\frac{1}{3}}| =\left| \frac{ (x^{\frac{1}{3}}-x_{n}^{\frac{1}{3}})(x^{\frac{2}{3}}+x^{\frac{1}{3}}x_{n}^{\frac{1}{3}}+x_{n}^{\frac{2}{3}})}{x^{\frac{2}{3}}+x^{\frac{1}{3}}x_{n}^{\frac{1}{3}}+x_{n}^{\frac{2}{3}}} \right| = \frac{|x-x_{n}|}{x^{\frac{2}{3}}+x^{\frac{1}{3}}x_{n}^{\frac{1}{3}}+x_{n}^{\frac{2}{3}}} \leq \frac{|x-x_{n}|}{x^{\frac{2}{3}}}.\\ \end{equation*}

Let \(\epsilon>0\text{.}\) Pick \(N\) so that \(n> N\) implies \(|x-x_{n}|\lt \epsilon x^{\frac{2}{3}}\text{.}\) Then the above implies that for \(n> N\text{,}\)

\begin{equation*} |x^{\frac{1}{3}}-x_{n}^{\frac{1}{3}}| \leq \frac{|x-x_{n}|}{x^{\frac{2}{3}}} \lt \frac{\epsilon x^{\frac{2}{3}}}{x^{\frac{2}{3}}}=\epsilon \end{equation*}

which thus proves \(x_{n}^{\frac{1}{3}}\rightarrow x^{\frac{1}{3}}\text{.}\)

Checkpoint 7.5.11.

Let \(a_n = \left( 1- \frac{1}{n^2}\right)^n\text{.}\) Show that \(a_n \to 1\) as \(n \to \infty\text{.}\) What happens if we consider \(\left(1-\frac{1}{n^s}\right)^n\) for \(s > 1\text{?}\) (Hint: Bernoulli's inequality.)

Solution.

By Bernoulli's inequality with \(h= -1/n^2\) we have

\begin{equation*} 1 \geq \left( 1- \frac{1}{n^2}\right)^n \geq 1 - \frac{n}{n^2} = 1 - \frac{1}{n}\text{.} \end{equation*}

By the squeeze theorem we deduce that

\begin{equation*} \left( 1- \frac{1}{n^2}\right)^n \to 1 \end{equation*}

as \(n \to \infty\text{.}\) A similar argument works for \(\left(1-\frac{1}{n^s}\right)^n\) whenever \(s > 1\text{.}\)

Checkpoint 7.5.12.

Let \(a_n = \left( 1+ \frac{1}{n^2}\right)^n\text{.}\) Show that \(a_n \to 1\) as \(n \to \infty\text{.}\)

Solution.

We have

\begin{equation*} \left( 1+ \frac{1}{n^2}\right)^n\left( 1- \frac{1}{n^2}\right)^n=\left( 1- \frac{1}{n^4}\right)^n\text{.} \end{equation*}

Let \(b_n = \left( 1- \frac{1}{n^2}\right)^n\) and \(c_n = \left( 1- \frac{1}{n^4}\right)^n\) so that \(a_n b_n = c_n\text{.}\) But by the previous exercise, \(b_n \to 1\) and \(c_n \to 1\text{.}\) Therefore by the rules for limits, \(a_n \to 1\) too.

Checkpoint 7.5.13.

Let \(a_n = \left( 1+ \frac{1}{n}\right)^n\text{.}\) Given that \(a_n \to L\) as \(n \to \infty\text{,}\) what can you say about the sequence \((b_n)\) where \(b_n = \left( 1- \frac{1}{n}\right)^n\text{?}\)

Solution.

By the argument for the previous exercise, \(b_n \to 1/L\) (since \(L \geq 1\)).
